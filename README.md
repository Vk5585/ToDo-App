A simple To-Do List application built in Java, allowing users to add, manage, and organize their daily tasks with a clean and minimal interface.
​
Overview

Task management app implemented in Java, ideal for practicing GUI and data handling concepts.
​
Lets users create, view, and manage a list of tasks in one place.
​
Designed as a beginner-friendly project to strengthen core Java and OOP skills.
​

Features

Add new tasks through an input field and button.
​
Mark tasks as completed or remove them from the list.
​
Simple and readable layout for easy task tracking.
​
Lightweight, fast, and easy to modify or extend.
